### SCSS

- base : mixin, variable 등 다른 scss에서 활용할 수 있는 기능을 작성

- components : 각 컴포넌트별 scss 작성

- css 폴더에 페이지 이름과 같은 scss 파일 생성 후, base와 components에서 import해서 사용

  - 이 scss 파일은 각 컴포넌트가 아닌 페이지에서 한꺼번에 import해서 사용
